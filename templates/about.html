🌱 About BioLoop AI

BioLoop AI is an AI-powered web application that estimates the value generated from organic waste.

Users enter the waste type, quantity, and location. The system predicts:

  • 🌿 Estimated CBG Yield
  • 💰 Estimated Revenue
  • 🌱 Organic Slurry Production

This project promotes sustainable waste management and supports the Waste to Wealth initiative using Artificial Intelligence.
